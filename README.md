Todo UI Home Page

## Assumption

- Frontend server runs on <http://localhost:3000>
- RESTAPI server runs on <http://localhost:8080>

## UI Deploy
- Open terminal.
- Type `npm install`
- Type `npm start`
- Go to <http://localhost:3000>, you're done!

- Follow Redux Framework
